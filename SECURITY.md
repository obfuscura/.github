# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any Obfuscura open-source package, please report it responsibly.

**Email:** [security@obfuscura.com](mailto:security@obfuscura.com)

Please include:

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Any potential impact assessment

We will acknowledge receipt within 48 hours and provide an initial assessment within 5 business days.

## Scope

This policy covers all repositories under the [obfuscura](https://github.com/obfuscura) GitHub organization.

For vulnerabilities in the Obfuscura web application or API, please refer to our [security.txt](https://obfuscura.com/.well-known/security.txt).

## Disclosure

We follow coordinated disclosure. We ask that you do not publicly disclose the vulnerability until we have had a reasonable opportunity to address it.
